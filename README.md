# lf-who-was-here
Extension phpBB3.2 - LF who was here?

# Installation:
* Copy the entire folder `bb3mobi/` with all subfolders and files to `ext/` 
* Navigate in the ACP to Customise -> Extension Management.
* Click Enable for "LF who was here".
